# Pose_estimation

This project focuses on pose estimation, where the goal is to predict the position and orientation (translation + rotation) of an object from input data.
It is designed for applications such as space robotics, satellite navigation, and computer vision tasks.

📊 Output

Predicted pose values (translation, rotation)

Model performance metrics
